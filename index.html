QDAB

QDAB is a simple Dots and Boxes AI. It is written in golang and python2.7. The basic idea is straightforward, UCT plus ANN.
QDAB can reach a high level. I compared QDAB with dabble.exe(http://www.mathstat.dal.ca/~jpg/dabble/) and PRsBoxes.exe(http://www.dianneandpaul.net/PRsBoxes/), as a result QDAB won in most cases.

Running

I have compiled QDAB for Windows and Linux users, please download correct version. You can find it in directory release.

Note: Please make sure you have seen "Server is running at 127.0.0.1:12345" after QDAB starts and before any other operation.

Windows

Uncompress the .zip file, and double click Qdab.bat.

Linux

First, make sure you have already installed python2.7, gtk, and python-simplejson module.
Second, uncompress the .tar.bz2 file and make Qdab.sh executable.
Third, enter the directory containing Qdab.sh, and run ./Qdab.sh.

Compilation

If you are interested in this program, you can make some changes and compile your version.
For Linux:
1. Install FANN 2.2.0 and golang1.2.2 (MUST version 1.2). You can find the setup files in directory env.
2. Run ./install.sh to compile all modules which will make directory bin.
3. Simply copy directory AnnModel from my release version to your directory bin or train your own ANN models. You need to understand my code before training your own ANN models.
4. The binary files needed to run QDAB are bin/server and src/guiclient/guiclient.py. Please note pwd MUST be directory bin when you run the server.
