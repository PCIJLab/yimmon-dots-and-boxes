Qdab

Qdab is a simple Dots and Boxes AI. It is written in golang and python2.7. The basic idea is straightforward, UCT plus ANN. Qdab can reach a high level. I compared Qdab with dabble.exe(http://www.mathstat.dal.ca/~jpg/dabble/) and PRsBoxes.exe(http://www.dianneandpaul.net/PRsBoxes/), the result is Qdab won them in most cases.

Running

I have compiled Qdab for Windows and Linux users, please download correct version before running.
Note: Please make sure you have seen "Server is running at 127.0.0.1:12345" after run Qdab and before any other operation.

Windows

Uncompress the .zip file, and double click Qdab.bat.

Linux

First, make sure you have already installed python2.7, gtk, and python-simplejson. Second, uncompress the .tar.bz2 file and make Qdab.sh executable. Finally, go to the directory containing Qdab.sh, run ./Qdab.sh.
